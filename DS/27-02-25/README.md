# INTRODUÇAO A JAVA

## Classe:

> Uma classe define um **tipo** de objeto. Ela pode determinar os seguintes dados:
- Atributos:
  - `int rm;`
    `private String nome;`
    `String email;`

- Construtores:
  - `public Aluno(int rm, String nome, String email) {<br>
        this.rm = rm;<br>
        this.nome = nome;<br>
        this.email = email;<br>
    }`

- Métodos:
  - `public void setNome(String nome) {<br>
        this.nome = nome.toUpperCase();<br>
    }`
