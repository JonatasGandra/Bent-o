# INTRODUÇAO A JAVA

## Classe:

> Uma classe define um **tipo** de objeto. Ela pode determinar os seguintes dados:
- Atributos:
  - `int rm;\
      private String nome;\\
      String email;`

- Construtores:
  - `public Aluno(int rm, String nome, String email) {\
        this.rm = rm;\
        this.nome = nome;\
        this.email = email;\
    }`

- Métodos:
  - `public void setNome(String nome) {\
        this.nome = nome.toUpperCase();\
    }`
